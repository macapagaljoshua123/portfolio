## ✨ Features

- 🎨 **Beautiful UI** - Clean, professional design with smooth animations
- 🌙 **Dark Mode** - Toggle between light and dark themes with galaxy/space theme
- 📱 **Fully Responsive** - Works perfectly on all devices
- 🚀 **Performance Optimized** - Lazy loading images, optimized bundle
- 📧 **Contact Form** - Integrated with Formspree for message handling
- 🎯 **Interactive Carousel** - Featured projects with clickable modal details
- 🔍 **Scroll Spy** - Active navigation highlighting
- ⬆️ **Back to Top** - Smooth scroll button after scrolling 300px
- ♿ **Accessible** - WCAG AA compliant color contrast, proper ARIA labels

## 🏃‍♂️ Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/macapagaljoshua123/portfolio.git

# Navigate to project directory
cd portfolio

# Install dependencies
npm install

# Start development server
npm run dev
